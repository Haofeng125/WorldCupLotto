# 世界杯彩票排行榜 ⚽️🏆

高中同学世界杯竞猜记账网站。每人初始 **50 元**，买彩票、串关投注，看谁笑到最后。

## 技术栈
- 后端：Node.js + Express + SQLite（单文件数据库）+ JWT
- 前端：React + Vite + Tailwind（世界杯主题，手机/电脑自适应）
- 部署：后端进程同时托管前端静态文件，**单进程**即可上线

## 功能
- **注册 / 登录**：注册需邀请码 `rugen`
- **大厅**：实时排行榜 + 我的资产卡片 + 导航按钮
- **添加投注**：一张票可加无限场（串关）；每场选 2 队、日期、玩法（胜/平/负/让胜/让平/让负/其他，可多选）+ 让球个数；填本金。余额不足无法下注。
- **我的**：历史投注、改/删、自助结算（中/未中 + 中奖金额）、重置密码、退出登录
- **管理后台**（仅管理员）：管理用户（改名/改密/删除）、管理任意投注（结算/改/删）

### 排行榜算法
> 总分 = 现金余额 + 未结算投注本金 + 已中奖金
- 下注：本金从现金扣除，进入「未结算」桶（总分不变）
- 未中：本金消失，总分下降
- 中奖：中奖金额计入现金余额

## 本地运行
```bash
npm run install:all   # 安装前后端依赖
npm run build         # 构建前端
npm start             # 启动，访问 http://localhost:3000
```

### 开发模式（热更新）
```bash
npm run dev:server    # 终端 1：后端 :3000
npm run dev:client    # 终端 2：前端 :5173（已配置 /api 代理）
```

## 部署提示
- 数据库为 `server/data.sqlite`，备份直接复制此文件即可。
- 生产环境建议设置环境变量 `JWT_SECRET`（自定义密钥）和 `PORT`。
- 球队名单在 `client/src/teams.js`，可随赛程随时增删。
